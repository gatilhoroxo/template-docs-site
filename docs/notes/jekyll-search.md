# Jekyll

Um SSG (Static Site Generator) bastante usado. Composto por Liquid Templating Language, YAML e Markdown (e possivelmente um pouco de html e css também). O jekyll pega os arquivos '.md' os transforma em 'html', 'css' e 'js' de acordo com as configurações do frontmatter em 'yaml', com as partes liquid e arquivos de configuração '.yml'.
